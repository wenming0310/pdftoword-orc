# PDF转Word转换器 - 手动测试指南

如果批处理文件无法正常运行，请按照以下步骤手动测试。

## 步骤1：打开命令提示符

1. 按 `Win + R`，输入 `cmd`，回车
2. 进入项目目录：
   ```
   cd /d D:\zwm\python\PDF转Word带界面程序
   ```

## 步骤2：测试Python环境

在命令提示符中运行：
```
python --version
```

**期望结果**：
```
Python 3.x.x
```

**如果出错**：
- 检查Python是否安装
- 检查是否添加到系统PATH

## 步骤3：测试虚拟环境

如果项目中有venv文件夹，运行：
```
venv\Scripts\activate
```

**期望结果**：
- 命令行前出现 `(venv)` 标识

## 步骤4：测试依赖库

逐个运行以下命令，检查是否有错误：

```
python -c "import PyQt5; print('PyQt5 OK')"
python -c "import fitz; print('PyMuPDF OK')"
python -c "import pdf2docx; print('pdf2docx OK')"
python -c "import pytesseract; print('pytesseract OK')"
python -c "import PIL; print('Pillow OK')"
python -c "import docx; print('python-docx OK')"
python -c "import cv2; print('OpenCV OK')"
python -c "import numpy; print('NumPy OK')"
```

**如果某个库报错**：
运行以下命令安装：
```
pip install -r requirements.txt
```

## 步骤5：测试Tesseract配置

运行：
```
python -c "import pytesseract; print(pytesseract.pytesseract.tesseract_cmd)"
```

**期望结果**：
```
D:\Program Files\Tesseract-OCR\tesseract.exe
```

**如果路径不对**：
编辑 `pdf_to_word_gui.py` 和 `formula_processor.py`，修改第12行和第5行：
```python
pytesseract.pytesseract.tesseract_cmd = r'D:\Program Files\Tesseract-OCR\tesseract.exe'
```

## 步骤6：测试Tesseract是否可用

运行：
```
python -c "from PIL import Image; import pytesseract; img = Image.new('RGB', (100, 50)); text = pytesseract.image_to_string(img); print('OCR OK')"
```

**如果出错**：
1. 检查 `D:\Program Files\Tesseract-OCR\tesseract.exe` 是否存在
2. 打开文件资源管理器，检查该路径

## 步骤7：测试主程序导入

运行：
```
python -c "import pdf_to_word_gui; print('主程序 OK')"
```

**如果出错**：
- 复制完整的错误信息
- 检查是否所有依赖都安装成功

## 步骤8：运行主程序

如果以上测试都通过，运行：
```
python pdf_to_word_gui.py
```

## 常见问题

### 问题1：批处理文件闪退

**原因**：批处理文件中的命令有错误

**解决方案**：
- 不要双击运行bat文件
- 右键选择"用命令提示符打开"或"编辑"
- 或者手动在命令提示符中运行命令

### 问题2：Python不是内部命令

**解决方案**：
1. 重新安装Python
2. 安装时勾选 "Add Python to PATH"
3. 重启电脑

### 问题3：模块导入失败

**解决方案**：
```bash
pip install -r requirements.txt
```

### 问题4：Tesseract路径不存在

**检查步骤**：
1. 打开文件资源管理器
2. 导航到 `D:\Program Files\`
3. 查看是否有 `Tesseract-OCR` 文件夹
4. 如果没有，安装Tesseract OCR

### 问题5：程序启动后立即崩溃

**解决方案**：
1. 记录完整的错误信息
2. 复制错误粘贴给我
